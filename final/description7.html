Childhood Mental Disorders Interface Prototype


This project was based off of my ADHD infographic poster. Working with my partner, we designed this website to be used by parents or anyone interested in learning more about a specific childhood mental disorder. The interface features a search button, news section, disorders drop down for a selection of the most common, and a care section to search for care in the viewers area. Each disorder's page would also feature a timeline displaying demographic and case studies.
